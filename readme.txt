Hello, here i am deploying a static html fle using github.
1) Go to Settings on the top of the github taskbar
2) on the right hand side scroll to Pages section.
3)Under the build and deployment section select Github Actions/ Deploy from a branch. Here Select Github Actions.
4) Select static html workflow and commit the static.yml file that is displayed after the selection.
5) Go to Actions on top bar on github and select the workflow to get the website or go to Pages under settings where we had deployed
the website , you will see the link to website.
6)"For Static HTML we are doing" copy the link and paste on the browser to see the deployed file, if the name of the file is index.html as it is the default file that git will look.
If the name is not index.html, then add "/filename.html" to the same link to view the deployed static html file.

Ex: gitlink/filename.html
